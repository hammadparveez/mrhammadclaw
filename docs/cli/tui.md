# `mrhammadclaw tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
mrhammadclaw tui
mrhammadclaw tui --url ws://127.0.0.1:18789 --token <token>
mrhammadclaw tui --session main --deliver
```
